# Couchbase Capella Provider 1.7.1: Upgrade and Information Guide

Here is a list of what's new in 1.7.1

## New Features

* Allow users to configure the global API request timeout by setting env var `CAPELLA_GLOBAL_API_REQUEST_TIMEOUT`

## Bug Fixes

* The http request timeout is 5 mins now to allow for long running operations such as VPC Peering.


## Changes

There are no deprecations as part of this release.

1.7.1 includes new features and general improvements. See the [CHANGELOG](https://github.com/couchbasecloud/terraform-provider-couchbase-capella/blob/master/CHANGELOG.md) for more specific information.
