# Couchbase Capella Provider 1.7.2: Upgrade and Information Guide

Here is a list of what's new in 1.7.2

## New Features

* Added support for cluster-stats datasource
* - Add acceptance test for the Log Streaming data source
- Add data source to provider

## Bug Fixes

* - update skill to implement either 1 or 2 datasources depending on the prompt
- better error handling in the tf-gen script
* Please include a summary of the fix/feature/change, including any relevant motivation and context
* We throw an error while succeeding to change single to multi nodes
* App Endpoint state field should not have useStateForUnknown set
* - Create a function which sets a string attribute with required, requires replace, and length of 1 validation
* - Fix import for App Service Log Streaming Activation Status resource
- Add documentation for App Service Log Streaming Activation Status resource
* - Use types
* Documentation for App Service Log Streaming resource

## Changes

1.7.2 includes new features and general improvements. See the [CHANGELOG](https://github.com/couchbasecloud/terraform-provider-couchbase-capella/blob/master/CHANGELOG.md) for more specific information.
